# BIAZ Admin Dashboard

> Frontend interface for BIAZ financial platform administration

This repository contains the CI/CD pipeline configuration for the BIAZ Admin Dashboard.

## CI Pipeline

The GitHub Actions workflow includes:
- **Lint & Test**: Code quality checks and test execution
- **Build**: Multi-environment builds (dev, staging, production)  
- **Security Scan**: Dependency vulnerability checking
- **Lighthouse CI**: Performance and accessibility testing
- **Docker Build**: Container image creation and registry push
- **Deployment**: Automated staging and production deployments
